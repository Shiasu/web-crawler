# WEB CRAWLER/SCRAPER. Получение необходимых данных на примере сайта www.synapsenet.ru (В примере получаются контактные данные на основе ИНН орг-ии).
Для запуска необходимо установить Node.js (c оф. сайта), затем используя Node.js из папки проекта установить все модули (командой npm install).
После окончания загрузки всех пакетов можно запускать проект, для этого также из Node.js (находясь в папке проекта) прописать команду node --harmony index.js.
Для изменения данных поиска, в файле index.js в массив arrOfNums необходимо добавить искомые номера. (В ближайшее время будет добавлена возможность обработки исходных данных из файла)
P.S. При изменении кода программы не рекомендуется убирать (цепочки промисов/асинхронность/ожидание завершения процедуры перед следующим запросом), дабы не DoSить сайт :)
